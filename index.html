통합 도서 검색 설정
알라딘, 네이버, 구글의 도서 오픈 API 데이터를 동시에 조회하여 책 정보를 한층 더 풍성하게 채워 넣습니다.

알라딘 OpenAPI 페이지에서 발급받은 TTBKey를 정확히 입력하세요.

네이버 개발자 센터에서 발급받은 애플리케이션 Client ID를 입력하세요.

네이버 개발자 센터에서 발급받은 Client Secret 키를 입력하세요.

구글 북스 API 키를 입력하세요. API 키가 없어도 일부 무료 한도 내에서 기본 검색이 가능합니다.

로컬 분석 실패 시 제미나이 AI를 활용해 책 본문에서 ISBN을 역추적합니다. (Google AI Studio에서 무료 발급 가능)

체크 시, 검색어 단어 목록이 수집된 도서 제목 내에 실제로 정확하게 포함되어 있는 결과만 통과시킵니다.

체크 시, 매뉴얼 검색창을 열었을 때 실제 도서의 표지 전후방 장들을 열어 판권지 속 ISBN을 추적합니다. (저사양 컴퓨터는 해제를 권장합니다.)
