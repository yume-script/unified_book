톻합 도서 검색 플러그인 : Unified BOOK Search (V20260715)
BookOasis의 메타데이터 검색을 한 차원 업그레이드해주는 통합 도서 검색 플러그인을 소개합니다! 알라딘, 네이버, 구글의 방대한 DB를 하나로 합쳐 가장 정확하고 선명한 도서 정보를 찾아줍니다.

주요 특징 (Key Features)
🔍 3-in-1 통합 검색: 알라딘, 네이버, 구글 도서 API를 동시에 검색하고 중복된 결과는 스마트하게 제거하여 보여줍니다.
🖼️ 고해상도 커버 수집: 썸네일 수준의 흐릿한 이미지는 NO! 각 API의 파라미터를 분석하여 시스템이 허용하는 최고 화질의 원본급 커버를 우선적으로 가져옵니다.
⚡ 스마트 WebP 변환: 다운로드한 이미지는 서버에서 즉시 WebP 포맷으로 최적화 저장되어, 용량은 줄이고 로딩 속도는 높였습니다.
🎯 엄격한 매칭 모드: 검색어와 제목이 일치하는 결과만 필터링하는 기능을 지원하여 검색 정확도를 높일 수 있습니다.
🏷️ 출처 표시 시스템: 검색 결과 리스트에서 알라딘, 네이버, 구글 등의 출처를 구분해 직관적인 선택이 가능합니다.

업데이트 내역 (V20260715)
DB 스키마 완벽 대응: release_date, summary, link 등 BookOasis 최신 스키마에 맞춰 데이터가 누락 없이 기록됩니다.
커버 화질 개선 로직: cover500 치환(알라딘), zoom=3 요청(구글), 사이즈 제한 제거(네이버) 로직 적용.
HTML 태그 정화: 설명문(Summary)에 포함된 불필요한 HTML 태그를 자동으로 제거하여 깔끔한 텍스트만 저장합니다.

설정 방법 (Configuration)
알라딘: TTBKey (필수 권장)
네이버: Client ID / Secret (필수 권장)
구글: API Key (선택 사항)
