# 📚 Unified BOOK Search (V20260715)

**BookOasis의 메타데이터 검색을 한 차원 업그레이드해주는 통합 도서 검색 플러그인입니다.**
알라딘, 네이버, 구글의 방대한 DB를 하나로 합쳐 가장 정확하고 선명한 도서 정보를 찾아줍니다.

![Main Banner/Demo](이미지주소_여기에_넣으세요)

---

## ✨ 주요 특징 (Key Features)

| 기능 | 설명 |
| :--- | :--- |
| **🔍 3-in-1 통합 검색** | 알라딘, 네이버, 구글 도서 API를 동시에 검색하고 중복 결과를 스마트하게 제거합니다. |
| **🖼️ 고해상도 커버 수집** | 저화질 이미지는 이제 그만! 시스템 허용 최고 화질의 원본급 커버를 우선적으로 가져옵니다. |
| **⚡ 스마트 WebP 변환** | 다운로드 이미지를 WebP로 최적화하여 서버 용량은 줄이고 로딩 속도는 극대화했습니다. |
| **🎯 엄격한 매칭 모드** | 검색어와 제목 일치 필터링 지원으로 검색 정확도를 비약적으로 높였습니다. |
| **🏷️ 출처 표시 시스템** | 검색 결과 리스트에서 알라딘/네이버/구글 출처를 직관적으로 확인하고 선택할 수 있습니다. |

---

## 🛠️ 업데이트 내역 (V20260715)

*   **DB 스키마 완벽 대응:** `release_date`, `summary`, `link` 등 BookOasis 최신 스키마에 맞춰 데이터 누락 없이 기록.
*   **커버 화질 개선:**
    *   알라딘: `cover500` 파라미터 치환
    *   구글: `zoom=3` 요청 적용
    *   네이버: 사이즈 제한 제거 로직 적용
*   **HTML 태그 정화:** 설명문(Summary) 내 불필요한 HTML 태그를 자동 제거하여 깔끔한 텍스트 저장.
